===

Create a greeter app that displays a greeting like:

```
<div>
  <span>{salutation}</span>
  <span>{name}</span>!
</div>
```

It also has an editor component that has inputs with two actions:

1. Update Salutation
2. Update Name

## Rubric *10pts*
- provider/store setup in index *2pt*
- reducer *2pt*
- actions *2pt*
- constants *2pts*
- containers *2pt*
